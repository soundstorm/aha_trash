# aHA Müllabholung – aha Zweckwirtschaft Hannover

Müllabfuhrkalender für die Region Hannover

[![HACS](https://img.shields.io/badge/HACS-Custom-41BDF5.svg)](https://my.home-assistant.io/redirect/hacs_repository/?owner=soundstorm&repository=aha_trash&category=integration)

Durch den Konfiguations-Flow ist die Einrichtung eigentlich recht unmissverständlich.
Zunächst Gemeinde auswählen, dann werden automatisch die Straßen zur Gemeinde geladen.
Dazu dann die entsprechende Hausnummer und ggf. Zusatz eingeben.
Eventuell wird ein Ladeort nötig, dieser kann im letzten Schritt ausgewählt werden.
